# RechercherBoissonIdeal
Moteur de recherche sémantique &amp; RAG 100% local appliqué aux boissons bien-être. > Une architecture 3-Tier associant SQL Server, Python et Streamlit. L'application utilise Ollama (nomic-embed-text et llama3) pour comprendre l'intention de l'utilisateur et générer des réponses adaptées sans aucune API cloud payante.
